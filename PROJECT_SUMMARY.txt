✨ KEY FEATURES IMPLEMENTED:

   ✅ Hierarchical 4-Layer Experimental Design
      Layer 1: Political vs Non-Political layer localization
      Layer 2: Left/Right/Neutral bias analysis within political layers
      Layer 3: Topic-level fine-grained analysis
      Layer 4: Steering intervention + multi-layer evaluation

   ✅ Multiple Analysis Methods
      • Cosine similarity gap (Safety Layers adapted)
      • Weak classifier probing (SVM + MLP per layer)
      • Neutral anchor triangulation (N-L, N-R, L-R)
      • Content vs Style decomposition (Bang et al.)
      • Inference-time steering (CAA method)

   ✅ Comprehensive Evaluation
      • Representation layer: bias_direction → 0?
      • Behavioral layer: generation stance changes
      • Capability layer: QA performance preservation
